Código para fazer a ligação com o banco de dados.
Colasse na classe java chamada "dao" e geralmente tem o nome de "ModuloConexao".


public static Connection conectar(){
        Connection conexao;
        
        String driver = "com.mysql.cj.jdbc.Driver";
        String url = "jdbc:mysql://localhost:3306/"NOME DA CONEXÃO NO BD"";
        String user = "root";
        String password = "";
        
        try {
            Class.forName(driver);
            conexao = DriverManager.getConnection(url, user, password);
            return conexao;
            
            } catch(ClassNotFoundException | SQLException e){
                System.out.println(e);
                return null;
            }
        
    }

Usa-se o import java.sql.* para importa tudo da biblioteca sql.
